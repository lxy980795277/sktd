# SKTD 官网项目说明

本文档概括 **SKTD（公司）** 的业务定位，以及 **本仓库中的网站** 在做什么；业务事实与里程碑以站内多语言文案（`src/i18n/content.ts`）为准，便于与线上展示保持一致。

---

## 网站是什么、解决什么问题

本仓库实现的是 **SKTD 的企业官网首页**：基于 **Next.js（App Router）** 与 **Tailwind CSS** 的多语言站点，默认提供 **英语（en）、德语（de）、中文（zh）** 路由（如 `/en`、`/de`、`/zh`）。

网站目标是把 SKTD 的 **跨境 B2B 价值主张** 结构化呈现：优势说明、行业覆盖、合作流程、客户证言区（当前为高质量占位文案，可后续替换为正式授权内容）、关于我们、五条服务能力、发展里程碑与联系入口（CTA）。

---

## SKTD 是做什么的（业务）

**SKTD** 是一家以 **连接亚洲与欧洲** 为核心使命的商业服务机构，强调 **可靠、端到端可控的执行**。

- **成立与根基**：2014 年于德国 **杜塞尔多夫** 成立，将德国工程严谨性、意大利设计文化与国际贸易实务结合，服务希望做跨境增长的品牌与企业客户。
- **地理与团队**：在 **杜塞尔多夫、米兰、上海、胡志明市** 设有战略办公室，支持多时区协调与多语言商务沟通。
- **端到端能力（五条服务线）**：
  1. **采购（Sourcing）**：围绕市场匹配度识别供应商与产品机会。
  2. **产品开发（Product Development）**：把商业目标落实为可制造、可上市的规格与方案。
  3. **质量管理（Quality Management）**：让检验与标准进入流程，而非事后补救。
  4. **项目融资（Project Financing）**：支撑跨境项目下的现金流与执行窗口。
  5. **物流（Logistics）**：协调运输、包装与交付节奏，提升计划性与可见性。

**典型服务场景**（与首页「行业与客群」模块一致）：从 **室内与家居、餐饮、零售、礼品、商务与酒店、生活方式** 等方向，为 B2B 客户提供可控的跨境产品与合作链路。

## 发展时间线（概要）

里程碑条目定义在 `src/i18n/content.ts` 的 `milestones.items` 中；**各语言文案不完全相同**，下表为各版本共有的主干事实（中/英/德对齐部分）。

| 年份     | 事件概要                                                                                                             |
| -------- | -------------------------------------------------------------------------------------------------------------------- |
| **2014** | 在德国杜塞尔多夫成立，以连接亚洲与欧洲为方向。                                                                       |
| **2015** | 年销售额达到约 **100 万欧元** 量级的首个重要节点。                                                                   |
| **2016** | 在中国上海建立战略采购 / 采购中心。                                                                                  |
| **2017** | 在意大利米兰扩展设计与销售相关布局。                                                                                 |
| **2019** | 强化东南亚布局（越南办公室）；并与 **KUKA HOME** 等建立战略合作（英文条目标题即写「越南办公室与 KUKA HOME 合作」）。 |
| **2022** | 启动 **低碳、减塑与可持续** 相关方向，纳入路线图。                                                                   |

**仅中文站文案额外写明的信息**（若对外统一口径，建议三语同步或删改以避免不一致）：2016 年「成为 **JYSK** 官方供应商」；2017 年「年销售额突破 **1000 万欧元**」。英文、德文当前里程碑描述较概括，未包含上述两点。

页面视觉上的时间标注（如「2014 → 2022」）用于概括展示区间，**具体叙事以各语言里程碑列表为准**。
